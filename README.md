**Selenium automation framework with JAVA**


#### Tech Stack
- Java 
- Selenium - 4.27
- TestNG
- Maven
- Page Object Model, Page Factory - Concepts.
- Apache POI - Excel
- Allure Reporting - HTML Reports
- Log4j - Logs
- Thread Local - Support - Parallely
- Listeners( Retry Listeners)
- Sonarlint - linting( Code Checks)
- Git -> Jenkins
